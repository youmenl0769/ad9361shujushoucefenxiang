       - **可调谐通道带宽**：支持带宽从小于200 kHz到高达56 MHz的配置，适合从窄带到宽带的应用。

         - **双通道接收器**：能够同时处理6路差分或12路单端输入信号，提高了接收端的多路复用能力和分析能力。

         **应用领域**
